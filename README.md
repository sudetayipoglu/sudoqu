# SudoQu

Yarışma/hackathon/fuar fırsatlarını otomatik keşfeden, takip eden ve ekip operasyonunu yöneten platform.

## Klasör Yapısı
- `backend/` — API, Fırsat Radarı, veritabanı mantığı
- `frontend/` — Kullanıcı arayüzü
- `docs/` — Proje dokümantasyonu

## Durum
V1 (Çekirdek MVP) geliştirme aşamasında.
